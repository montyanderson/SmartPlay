# yrs2015

## Install

* Install Node.js or io.js

* Clone the repository

* Install the dependencies

``` bash
npm install
```

* Run the app!

``` bash
node server
```

## Development

* Automatically compile browserify and less scripts/styles

``` bash
npm run dev
```

* Automatically restart the node server on file change

``` bash
npm install nodemon -g
nodemon server
```
